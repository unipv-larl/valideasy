## Requirements
- python3
- udapi package

## Usage

### Clone this repo

```bash
git clone https://github.com/unipv-larl/valideasy.git
```

### Move inside the directory

```bash
cd ./valideasy
```

### Run the script

```bash
python valideasy.py path/to/the/treebank
```
